This repository contains C programs developed as part of the **Data Structures with Algorithms Laboratory** course. Each program demonstrates the implementation of a fundamental data structure or algorithm.

## Program 1: Infix to Postfix Conversion

Implements a program in C to convert an infix expression to postfix using stack operations.

## Program 2: Postfix Expression Evaluation

Evaluates a valid postfix expression using a stack. The expression contains non-negative single-digit operands and binary arithmetic operators (+, -, *, /).

## Program 3: Queue using Array

Simulates the working of a queue using an array with the following operations:
- Insert
- Delete
- Display

## Program 4: Singly Linked List Operations

Implements the following operations on a singly linked list:
- Insert and display
- Delete from beginning/end
- Delete a specific element

## Program 5: Searching Techniques

Implements two search algorithms:
- Linear Search
- Binary Search

## Program 6: Sorting Algorithms

Implements two sorting algorithms using user-defined functions:
- Bubble Sort (Ascending Order)
- Selection Sort (Descending Order)

## Program 7: Kruskal's Algorithm

Finds the Minimum Cost Spanning Tree of an undirected graph using Kruskal’s algorithm.

## Program 8: Dijkstra's Algorithm

Finds the shortest path from a given vertex to all other vertices in a weighted connected graph using Dijkstra’s algorithm.

## Program 9: Polynomial Addition using Circular Representation

Accepts two polynomials using circular linked list representation, adds them, and prints the resulting polynomial.
